Auto New Tab

Choose the sites where the extension is active. On those sites, tapped links open in a new tab unless a rule says otherwise. Changes save automatically.

Active sites

Patterns match the page URL. * is a wildcard, e.g. https://example.com/account/*.

Link rules

When you tap a link on an active site, the first matching rule (top to bottom) decides. “Same tab” leaves the link alone. Links matching no rule use the default action.
